Site de Notícias sobre Android (Responsivo)

Este projeto foi iniciado com a criação de um wireframe para planejar a estrutura da página e a disposição dos elementos. A interface foi desenvolvida utilizando HTML e CSS, com aplicação de responsividade através de media queries, garantindo boa visualização em celulares e computadores. A paleta de cores foi definida com base em tons monocromáticos de verde, utilizando Adobe Color, e a tipografia foi escolhida para melhorar a hierarquia visual e a legibilidade do conteúdo.

Principais resultados:

DLayout organizado e intuitivo Adaptação para diferentes tamanhos de tela Melhor experiência do usuário na leitura de conteúdo Aplicação prática de conceitos de UI e estrutura web
